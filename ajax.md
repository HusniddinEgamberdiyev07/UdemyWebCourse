**JSON** - **J**ava**S**cript **O**bject **N**otation.

**JSON.parse()** -> turns json to js

**HTTP Verbs**

- get -> getting information
- post -> sending information
- put
- delete

**HTTP status codes**

- around 200 -> successful responses
- around 300 -> redirection messages
- around 400 -> client error responses
- around 500 -> server side error reponses
