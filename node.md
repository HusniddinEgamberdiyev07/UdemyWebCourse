## Terminal

Text based interface to your computer.

**shell** program running on terminal

### Commands

**1. ls**  - list. Shows all folders and files inside folder.

**2. pwd** - shows where you are. Example: /home/husniddin/WebUdemyCourse

**3. cd** - change directory.

**4. clear** - clears terminal.

**5. mkdir** - make directory.

**6. man** - manual.

**7. touch** - creates a new file

**8. rm** - removes file. It will not put it inside trash can. rm -rf -> removes directory with files.

**9. rmdir** - removes empty directory.

**cd ~** - Go home.

**cd /** - Go root.

**Absolute path** - /home/husniddin/WebUdemyCourse
Works anywhere

**Relative path** - WebUdemyCourse
Works only where you are

**flags** - starts with -. ls -l
